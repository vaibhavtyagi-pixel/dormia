# DORMIA — Sleep is now a sport.

Live demo: https://dormia-eight.vercel.app

> Built for Google Tech Roulette Challenge — Team 26, Fractional Experts

---

## What it is

Dormia is a competitive sleep sport. Set your nightly sleep target, 
hit it and climb the leaderboard, earn XP and grow your streak. 
Miss it and Gemini writes your public obituary for your entire 
league to see.

Sleep is the currency. The leaderboard belongs to whoever 
went to bed first.

---

## Features

- Google Sleep API — passive on-device sleep detection, no wearable needed
- Live leaderboard — My League, City, Country, Continent, World rankings
- Gemini-generated obituaries — public roasts when your streak dies
- League system — create or join private leagues with a shared code
- Improve tab — AI night plan generated by Gemini
- Android APK — integrates Sleep API confidence scores to detect sleep and wake

---

## Google APIs Used

- Google Sleep API
- Google Maps API
- Gemini API
- Firebase Realtime Database
- Firestore
- Firebase Authentication

---

## Stack

React + Vite, Firebase, Gemini API, Google Maps API, Vercel

## Run locally

git clone https://github.com/vaibhavtyagi-pixel/dormia
cd dormia
npm install
cp .env.example .env
# fill in your Firebase and Gemini keys
npm run dev
